# Feature Specification: 鼓風機入口法蘭 DWG 產生器

**Feature Branch**: `001-flange-generator`

**Created**: 2026-05-18

**Status**: Draft

**Input**: 詳見 `docs/req/req-1.md` v0.4；本規格為其業務面提煉版本。

---

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 業務即時產出法蘭預覽 (Priority: P1)

業務人員在與客戶互動（電話、會議、現場拜訪）時，需要在不依賴工程師、不切換軟體的前提下，
快速依客戶口述的法蘭規格產生一張可視化的工程圖，當場與客戶確認尺寸、孔位、外型。

**Why this priority**: 此故事即為產品存在的核心理由。沒有它，業務只能等工程師畫圖（一般 1–2 天），
客戶決策延遲；有它，業務可在通話/會議中當下完成規格確認，提升成單率與客戶滿意度。
此故事獨立完成即構成可發布的 MVP。

**Independent Test**: 業務人員在瀏覽器開啟工具 → 輸入內徑、PCD、外徑、孔數、孔徑、厚度六項數值
（材質鎖定 SS400）→ 點擊「產生」→ 在右側看到一張帶有英文標註的法蘭工程圖。
不需要任何下載、不需要安裝任何軟體即可完成這項驗證。

**Acceptance Scenarios**:

1. **Given** 業務人員開啟工具頁面，**When** 依序輸入有效的六個數值欄位並點擊「產生」，
   **Then** 頁面右側於 5 秒內出現一張法蘭工程圖，圖面以英文標註全部 7 個欄位的數值
   （含預設材質 SS400），且包含「For Customer Preview Only — Not for Manufacturing」浮水印。
2. **Given** 業務人員已產生預覽，**When** 任意修改某一欄位數值並再次點擊「產生」，
   **Then** 預覽圖即時更新為新規格的法蘭，**舊圖不被保留**，前後不混淆。
3. **Given** 業務人員輸入不合理的數值（例如 PCD 等於內徑），**When** 點擊「產生」，
   **Then** 系統明確顯示中文錯誤訊息指出哪一欄位、為何不合理，且**不**產生任何錯誤圖。

---

### User Story 2 - 提供客戶可攜的 DWG 與 PDF 檔案 (Priority: P1)

業務人員在客戶確認規格後，需要將同一份法蘭圖以兩種格式交付給客戶：
DWG 給有 AutoCAD 的客戶（如工程部門）、PDF 給沒有 AutoCAD 的客戶（如採購、業主、現場主管）。
這份檔案讓客戶離開會議後仍能複查、轉寄、列印、附入採購文件。

**Why this priority**: 預覽只解決「當下溝通」，下載才解決「後續流通」。若沒有此故事，
業務人員仍需請工程師另外出圖，工具的價值大打折扣。與 US1 同列 P1 因兩者構成完整 MVP。

**Independent Test**: 在 US1 預覽完成後，分別點擊「下載 DWG」與「下載 PDF」兩顆按鈕，
分別取得兩個檔案：DWG 可在 AutoCAD 中正常開啟、PDF 可在 macOS Preview / Chrome / Acrobat / iPad
中正常開啟，兩者內容皆對應預覽圖的規格。

**Acceptance Scenarios**:

1. **Given** 業務已產生預覽，**When** 點擊「下載 DWG」，**Then** 瀏覽器於 3 秒內開始下載，
   檔案為 AutoCAD R2000 格式，於 AutoCAD 開啟後可見完整法蘭工程圖與英文標註，
   且含浮水印。
2. **Given** 業務已產生預覽，**When** 點擊「下載 PDF」，**Then** 瀏覽器於 3 秒內開始下載，
   檔案為 A4 直式 PDF，第一頁含法蘭工程圖與英文標註，並附一份規格純表格列出全部 7 個欄位數值
   與 SS400 材質，含浮水印與產生時間。
3. **Given** 業務同一規格連續下載 DWG 兩次，**When** 比對兩份檔案的拓樸結構與主要尺寸欄位，
   **Then** 結構與尺寸完全一致（依 Constitution 原則 V 結構一致性）。
4. **Given** 業務修改了部分欄位但尚未重新產生預覽，**When** 點擊「下載 DWG」或「下載 PDF」，
   **Then** 下載結果反映**最新輸入的數值**，而非舊預覽的數值（即下載按下時為真）。

---

### Edge Cases

- **極小法蘭**：內徑 1 mm、外徑 5 mm、孔徑 0.5 mm —— 系統 MUST 接受並產生對應尺寸的圖；
  圖面標註不可彼此重疊到無法閱讀（標註位置自動避讓）。
- **極大法蘭**：外徑 5000 mm 等級 —— 系統 MUST 接受；PDF 仍為 A4 直式，圖形等比縮放至頁面範圍內，
  不自動切換 A3。
- **孔數 = 1**：系統 MUST 接受並只繪製一個孔（位於 0° 方向）。
- **孔數極多（如 64 孔）**：系統 MUST 接受並均布；標註可採代表性標註方式避免擁擠。
- **小數點第 3 位以上輸入**：前端 MUST 即時截斷至第 2 位並提示「已自動截斷」。
- **PCD ≤ 內徑** 或 **PCD ≥ 外徑**：系統 MUST 在送出前以中文訊息明確回拒。
- **孔徑過大導致孔之間重疊或孔超出內外徑邊界**：系統 MUST 以中文訊息回拒並指出具體幾何衝突。
- **使用者輸入材質欄位為非 SS400 值**：v1 此欄位於 UI 上鎖定為 SS400 不可改；若以 API
  直接傳入非 SS400 值，後端 MUST 回拒。
- **使用者於下載中關閉頁面**：因系統無狀態，無清理動作，下次重新開啟即可重新生成。
- **同時點擊「下載 DWG」與「下載 PDF」**：兩次請求 MUST 各自獨立完成，互不影響。

## Requirements *(mandatory)*

### Functional Requirements

**輸入與驗證**

- **FR-001**: 系統 MUST 提供一個 Web 表單，包含 7 個欄位：內徑、PCD、外徑、孔數、孔徑、厚度、材質。
- **FR-002**: 系統 MUST 將「材質」欄位於 v1 中鎖定為 SS400，使用者無法修改。
- **FR-003**: 系統 MUST 要求六個數值欄位（內徑、PCD、外徑、孔數、孔徑、厚度）皆為必填；
  缺漏任一項時，「產生」按鈕保持不可點擊狀態。
- **FR-004**: 系統 MUST 限制 mm 數值欄位最多 2 位小數，超出部分自動截斷並提示。
- **FR-005**: 系統 MUST 在送出前於前端驗證幾何合理性：內徑 < PCD < 外徑、孔徑 > 0、
  孔徑 < (外徑 − PCD)/2、孔徑 < (PCD − 內徑)/2、孔數 ≥ 1、所有 mm 值 > 0。
- **FR-006**: 系統 MUST 對任一驗證失敗以**繁體中文**訊息明確指出哪一欄位、為何不合理。
- **FR-007**: 後端 MUST 於收到請求時再次執行 FR-005 的全部驗證，不信任前端驗證結果；
  任一失敗以結構化錯誤回應，訊息採英文（圖面與後端錯誤訊息採英文，UI 提示採中文）。
- **FR-008**: 系統 MUST NOT 對缺漏欄位自動填入預設值（材質的 SS400 屬於 v1 鎖定值，非預設替代）。
- **FR-009**: 系統 MUST 統一在輸入區塊標題標示單位「mm」，不在每個欄位旁重複標示。

**產出**

- **FR-010**: 系統 MUST 於使用者點擊「產生」後 5 秒內，於頁面右側顯示一張法蘭 2D 工程圖預覽。
- **FR-011**: 預覽 MUST 包含全部 7 個欄位的數值標註，標註語系為**英文**。
- **FR-012**: 預覽 MUST 顯示浮水印「For Customer Preview Only — Not for Manufacturing」。
- **FR-013**: 系統 MUST 提供「下載 DWG」按鈕；點擊後於 3 秒內開始下載 AutoCAD R2000 格式檔案。
- **FR-014**: 系統 MUST 提供「下載 PDF」按鈕；點擊後於 3 秒內開始下載 A4 直式 PDF 檔案。
- **FR-015**: 下載的 DWG 圖面 MUST 包含全部 7 個欄位的英文標註與浮水印。
- **FR-016**: 下載的 PDF MUST 包含：（a）法蘭工程圖含英文標註；（b）一份純表格列出 7 個欄位數值；
  （c）浮水印；（d）產生時間。**不**包含公司 Logo、不採報價單版型。
- **FR-017**: 系統 MUST 採用以下檔名規則：
  `flange_{內徑}x{外徑}_PCD{PCD}_{孔數}H_{厚度}t_{timestamp}.{dwg|pdf}`
- **FR-018**: 系統 MUST 將第一孔位於 0°（+X 軸方向），其餘孔以 PCD 圓周均布。
- **FR-019**: DWG / PDF / SVG 預覽三者所呈現的法蘭規格 MUST 完全對應同一份使用者當下輸入。

**狀態與行為**

- **FR-020**: 系統 MUST 為完全無狀態：不持久化任何產生結果、不記錄使用者輸入歷史、
  不留下使用者識別資料。
- **FR-021**: 系統 MUST 在使用者修改輸入後重新點擊「產生」時，覆蓋現有預覽，**不**保留歷史。
- **FR-022**: 「下載 DWG」與「下載 PDF」MUST 以使用者當下表單值（而非預覽快取）為依據即時產生。
- **FR-023**: 系統 MUST 同時支援使用者連續或並發下載 DWG 與 PDF，兩者互不影響。

**介面與語系**

- **FR-024**: UI 介面語系 MUST 為繁體中文（含按鈕、欄位標籤、錯誤訊息、提示）。
- **FR-025**: DWG 與 PDF 圖面標註語系 MUST 為英文。
- **FR-026**: 系統 MUST 支援以下瀏覽器最新兩個主版本：Chrome、Edge、Safari。
- **FR-027**: UI MUST 於 1366×768 桌機與 iPad 直橫式（768×1024 / 1024×768）下可用。

### Key Entities

- **Flange Specification（法蘭規格）**：使用者輸入的完整規格集合。屬性：
  內徑、PCD、外徑、孔數、孔徑、厚度（皆 mm，小數 2 位）、材質（固定 SS400）。
  本實體不持久化，僅存在於單一請求生命週期內。
- **Flange Rendering（法蘭呈現）**：基於規格產生的可視化輸出，包含三種同源呈現：
  SVG 預覽、DWG R2000 檔、PDF A4 直式檔。三者皆無持久化儲存。

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 新業務人員首次使用本工具即可在 1 分鐘內完成第一張法蘭圖的產出
  （不需教學文件或同事指導）。
- **SC-002**: 從首次點擊「產生」到看見預覽圖，95% 的請求於 5 秒內完成。
- **SC-003**: 從點擊「下載」到瀏覽器開始下載檔案，95% 的請求於 3 秒內完成。
- **SC-004**: 業務向客戶口頭確認規格後，於同一通電話或同一場會議內即可交付 DWG / PDF
  （不再需要工程師另行繪圖等待）。
- **SC-005**: 客戶端不需要安裝 AutoCAD 即可看到法蘭規格（透過 PDF 達成）。
- **SC-006**: 同一規格在不同次數產出的 DWG / PDF，主要尺寸欄位（內徑、PCD、外徑、孔數、孔徑、
  厚度）100% 一致，可被客戶比對為「同一規格」。
- **SC-007**: 100% 的不合理輸入（缺漏或幾何不合理）在送出前或送出時被回拒，
  系統 0% 產生不合理的法蘭圖。
- **SC-008**: 100% 的產出（SVG 預覽、DWG、PDF）皆含「For Customer Preview Only —
  Not for Manufacturing」浮水印，且全部 7 個欄位皆有對應標註。

## Assumptions

- 使用者（業務人員）已習慣 Web 表單操作，對「內徑、PCD、外徑、孔數、孔徑、厚度」六個術語有基本認知；
  若未來資料顯示業務不熟悉術語，再考慮加入欄位旁說明圖示（不在 v1 範圍）。
- v1 工具僅在開發者本機 local 環境運行，不假設多人同時並發；單機效能足以支撐單一使用者。
- 客戶端有現代瀏覽器（Chrome / Edge / Safari 最新兩版）或 PDF 閱讀器（macOS Preview /
  Adobe Reader / iPad），這已是業務與客戶現場互動的常態。
- 第一孔角度沿用 Constitution 預設 0°（+X 軸方向）；待客戶端有實際反映再調整。
- v1 不需登入；本工具於公司內網 local 環境，無外部存取風險。
- 「無狀態」意味著重啟伺服器後不留任何檔案、不留任何使用者輸入紀錄；若日後需審計，
  將於後續階段補上記錄機制（非 v1 範圍）。
- SS400 為目前業務涵蓋的最主要材質；其他材質列入後續階段。
- 法蘭為對稱平面件，2D 工程圖足以表達規格給客戶，無 3D 預覽需求（依 Constitution 用途定位）。

## Out of Scope

本規格明確不包含下列項目（依 req-1.md §8 範圍外）：

- SS400 以外的材質
- 多階法蘭、異形法蘭、橢圓法蘭
- 3D 預覽、第三方 DWG viewer 整合
- DWG / PDF 圖面標註中文版本
- 第一孔起始角度的客製化設定
- 雲端、Docker、Electron 等部署方式
- 多人並行、使用者登入、帳號權限管理
- 歷史紀錄、模型版本控管、變更歷史查詢
- 自動套用 ASME / JIS / CNS 標準法蘭表
- 嚴格工程公差控管（待客戶提供公差表後升版 Constitution 觸發）
- 直接將模型送 CAM / 加工系統
- PDF 自動依尺寸切換紙張大小
- PDF 附公司 Logo、報價單版型、業務簽章
- 持久化儲存產生結果或使用者輸入
